# Taraxa区块链账户查询器 - 项目大纲

## 项目概述
创建一个功能完整的Taraxa区块链账户查询器，用户可以通过输入钱包地址快速查询账户余额，并获得清晰的可视化展示。

## 文件结构

### 核心HTML页面
- **index.html** - 主页面，包含地址输入、余额查询和结果展示
- **about.html** - 关于页面，介绍Taraxa区块链和查询器的功能
- **history.html** - 查询历史页面，显示用户的查询记录

### JavaScript文件
- **main.js** - 主要逻辑文件，包含RPC查询、数据处理和用户交互
- **effects.js** - 视觉效果和动画控制
- **utils.js** - 工具函数，如地址验证、数据格式化等

### 资源文件
- **resources/** - 存放图像、图标和其他静态资源
  - taraxa-logo.png - Taraxa标志
  - blockchain-bg.jpg - 区块链主题背景
  - icons/ - 各种图标文件

## 功能模块

### 1. 地址输入与验证模块
- 地址格式验证（0x开头的42位十六进制）
- 实时输入反馈
- 地址历史记录
- 剪贴板粘贴支持

### 2. RPC查询模块
- 连接Taraxa主网RPC节点
- 发送eth_getBalance请求
- 处理响应数据
- 错误处理和重试机制

### 3. 余额展示模块
- 大字体余额显示
- TARA单位转换
- 实时更新动画
- 余额历史图表

### 4. 网络状态监控
- RPC连接状态
- 网络延迟显示
- 最新区块信息
- 网络切换功能

### 5. 历史记录管理
- 本地存储查询历史
- 快速重新查询
- 历史数据可视化
- 数据导出功能

### 6. 视觉效果系统
- 动态背景粒子效果
- 查询加载动画
- 状态转换动画
- 响应式布局动画

## 技术实现

### 前端技术栈
- **HTML5** - 语义化标记
- **CSS3** - 响应式样式和动画
- **JavaScript ES6+** - 现代JavaScript特性
- **Tailwind CSS** - 快速样式开发

### 核心库依赖
- **Anime.js** - 动画库
- **ECharts.js** - 数据可视化
- **Shader-park** - 背景效果
- **PIXI.js** - 高性能图形渲染
- **Matter.js** - 物理动画
- **Splide.js** - 轮播组件

### RPC交互
- **JSON-RPC 2.0** - 标准RPC协议
- **eth_getBalance** - 查询余额方法
- **Taraxa主网节点** - https://rpc.mainnet.taraxa.io/
- **错误处理** - 完整的错误处理机制

## 用户体验流程

### 主要用户路径
1. 用户进入主页 → 看到清晰的查询界面
2. 输入钱包地址 → 获得实时验证反馈
3. 点击查询按钮 → 看到加载动画
4. 查看结果 → 获得清晰的余额信息
5. 查看历史 → 了解查询记录
6. 切换网络 → 测试不同环境

### 错误处理流程
1. 地址格式错误 → 友好提示和修正建议
2. 网络连接失败 → 重试选项和网络状态检查
3. RPC请求超时 → 超时处理和状态反馈
4. 无效地址 → 地址不存在提示
